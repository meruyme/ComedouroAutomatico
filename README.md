# ComedouroAutomatico
Final project of the Embedded System class. The project consists in a automatic pet feeder, utilizing Arduino. 

Utilized Components:

* Arduino Uno R3
* Various jumpers
* Vatious resistors
* 3 pushbuttoms
* 1 Display LCD 16x2
* 1 Servo Motor MG996R
* 2 Micro Servos 9g SG90 TowerPro
* 3 Protoboards 400 dots
